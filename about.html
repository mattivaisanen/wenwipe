About wenwipe

This is a basic site to check when the next rust wipe approximately is for everyone from new rust players to the most hardcore rust veterans.

How this site was created:

The site's design is based on Bootstrap 5 and basic CSS.
The timers were created using JavaScript with moment.js. A very large portion of the JavaScript was written by ChatGPT as i am horrible at JavaScript.
The site is opensource and can be viewed at github.com/mattivaisanen/wenwipe

This site was purely created to try out new stuff and learn some web development.
It is most defenitely not a passion project as Rust fucking sucks.
